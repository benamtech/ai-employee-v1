## Proof rules

- Scores and graph metrics are not test oracles.
- Every hard hyperedge has accepted proof, planned proof, or an explicit blocker.
- P1 certificate verification is formal-model evidence, not runtime evidence.
- P2 correspondence binds the model to repository facts.
- P3 advances only through exact executable evidence.
- P4 remains external until crossed.
- Unknown prerequisites remain blocked rather than mocked.
- Tests are never weakened for green.
