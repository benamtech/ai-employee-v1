The soul

Two type roles. Two color modes. One grid.

AMTECH is the 369 method — ×3 spacing, hairline composition, one hero per screen, no decoration — wearing an Ogilvy–Bloomberg skin: Inter for the voice, IBM Plex Mono for the machine, near-black on white and white on AMTECH red. Everything below is rendered in that system, live, with its real production values. If a choice isn’t on this page, it isn’t in the product.

Foundation 01

Color

Two text/background modes, one accent. Red is the only hue and it is spent only where the owner must look; everything else is ink, hairline, and wash.

Mode 01
Near-black on white

The reading surface. Red accents it: labels, the logo period, active borders, alerts, key CTAs.

Mode 02
White on AMTECH red

The commitment surface — the front-door band, a gated decision. White dominates; black may accent but never becomes the reading mode.

Foundation 04

Composition

Every screen is BASE → PILLARS → ROOF: pick the sections, pick the one hero, pick the primitive each section uses. Sections separate by a hairline border, never a whitespace gap.

The spinewindow bar → status → hero → body → footer
Window bar — identity + primary action
Status strip — segmented state
Hero — the one thing the eye lands on
dominates by size + isolation
Body — bordered sections, dense data in real tables
Footer — quiet hints
One hero per screen. Radius 0 everywhere. No shadow, gradient, blur, or icon library — text glyphs and hand-drawn pixel marks only.
