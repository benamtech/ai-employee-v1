# AMTECH Founder Logs

**Status: active**

This folder is for operational logs created while executing the 52-week AMTECH plan. The wiki explorer can write Markdown logs here when opened in a browser that supports local folder access. If the browser cannot write files directly, download the generated Markdown and place it in this folder.

## Naming convention

- `week-01.md`
- `week-02.md`
- `daily-YYYY-MM-DD.md`
- `customer-YYYY-MM-DD-name.md`
- `experiment-YYYY-MM-DD-name.md`

## Agent rule

Agents working inside this wiki should treat these files as operating evidence. Do not overwrite a founder log unless explicitly asked. Append new observations or create a new dated file.

## Weekly log fields

Each weekly log should include:

- revenue collected
- MRR
- new prospects
- emails sent
- calls made
- replies
- demos
- paid offers
- closed deals
- delivery hours
- reusable assets created
- biggest objection
- best customer quote
- what was learned
- next week's one experiment
- what to stop doing
